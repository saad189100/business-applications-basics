# Business Applications Basics

A Python basics booster course tailored for business students, covering practical programming concepts with real-world business examples.

## Live Site

https://saad189100.github.io/business-applications-basics/

## Files

- `python-booster-course.html` — the main course page (self-contained, no dependencies)
- `python-booster-session-outline.md` — session outline and planning notes

## Updating the Site

Edit `python-booster-course.html`, then:

```bash
git add python-booster-course.html
git commit -m "your message"
git push
```

GitHub Pages redeploys automatically within ~60 seconds.
